## ✨ 功能特性

- 🔗 **多链支持** - 以太坊(ETH/BSC/Polygon等)、波场(TRX)、Solana(SOL)
- 🔑 **双模式生成** - 助记词模式 / 私钥模式
- 🔄 **灵活匹配** - 共用模式(一套密钥多链) / 独立模式(各链独立)
- ⚡ **多进程并行** - 自动检测CPU核心数，充分利用算力
- 📊 **实时进度** - 显示速度、进度条、预估剩余时间
- 🛡️ **安全退出** - Ctrl+C 优雅停止，不丢失已找到的结果

## 📦 安装

### 环境要求

- Python 3.7+
- 支持 Windows / Linux / macOS

### 安装依赖

```bash
# 克隆项目
git clone https://github.com/YangyangX3/CryptoVanityMiner.git
cd CryptoVanityMiner

# 安装依赖
pip install -r requirements.txt
```

## 🚀 快速开始

```bash
python vanity_miner.py
```

可选安全参数：
- `--network-check passive|active|skip`：默认 `passive`（不外联）；`active` 会主动连接公网用于探测；`skip` 完全跳过检测
- `--no-clipboard`：禁用复制到剪贴板
- `--debug`：打印详细异常堆栈（可能包含敏感信息）

按照交互提示：
1. 输入目标尾号（如 `888`、`ABC`）
2. 选择生成模式（助记词/私钥）
3. 选择目标网络（ETH/TRX/SOL）
4. 选择匹配模式（共用/独立）
5. 等待生成完成

## 📖 使用指南

### 模式说明

#### 生成模式

| 模式 | 说明 | 速度 | 安全性 |
|-----|------|------|-------|
| 助记词 | 生成12词助记词 | ⭐⭐ | ⭐⭐⭐⭐⭐ |
| 私钥 | 直接生成随机私钥 | ⭐⭐⭐⭐⭐ | ⭐⭐⭐ |

#### 匹配模式

| 模式 | 说明 | 速度 | 结果 |
|-----|------|------|------|
| 共用 | 一套密钥满足所有选中网络 | 较慢 | 1套密钥 |
| 独立 | 每个网络单独生成 | 快 | N套密钥 |

### 概率参考

#### ETH地址 (16进制)

| 尾号长度 | 概率 | 8核预估时间 |
|---------|------|------------|
| 1位 | 1/16 | <1秒 |
| 2位 | 1/256 | <1秒 |
| 3位 | 1/4,096 | ~5秒 |
| 4位 | 1/65,536 | ~1分钟 |
| 5位 | 1/1,048,576 | ~20分钟 |
| 6位 | 1/16,777,216 | ~5小时 |

#### TRX/SOL地址 (Base58)

| 尾号长度 | 概率 | 8核预估时间 |
|---------|------|------------|
| 1位 | 1/58 | <1秒 |
| 2位 | 1/3,364 | ~5秒 |
| 3位 | 1/195,112 | ~4分钟 |
| 4位 | 1/11,316,496 | ~4小时 |
| 5位 | 1/656,356,768 | ~10天 |

> ⚠️ 共用模式下概率相乘，时间显著增加

### 字符限制

| 网络 | 有效字符 |
|-----|---------|
| ETH | `0-9`, `a-f` (不区分大小写) |
| TRX | `1-9`, `A-Z`, `a-z` (无0,O,I,l) |
| SOL | `1-9`, `A-Z`, `a-z` (无0,O,I,l) |

## 💡 使用建议

1. **尾号长度建议**
   - ETH: 3-4位较为实际
   - TRX/SOL: 2-3位较为实际

2. **模式选择**
   - 追求速度 → 私钥 + 独立模式
   - 追求统一管理 → 助记词 + 共用模式

3. **安全建议**
   - 在离线/断网环境运行
   - 生成后立即手抄保存
   - 切勿截图或网络传输
