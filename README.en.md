## English

### ✨ Features

- 🔗 **Multi-Chain Support** - Ethereum (ETH/BSC/Polygon), Tron (TRX), Solana (SOL)
- 🔑 **Dual Generation Modes** - Mnemonic Mode / Private Key Mode
- 🔄 **Flexible Matching** - Shared Mode (one key for all chains) / Independent Mode (separate keys)
- ⚡ **Multi-Process Parallel** - Auto-detect CPU cores for maximum performance
- 📊 **Real-time Progress** - Display speed, progress bar, and estimated time
- 🛡️ **Safe Exit** - Ctrl+C graceful stop without losing found results

### 📦 Installation

#### Requirements

- Python 3.7+
- Supports Windows / Linux / macOS

#### Install Dependencies

```bash
# Clone the repository
git clone https://github.com/YangyangX3/CryptoVanityMiner.git
cd CryptoVanityMiner

# Install dependencies
pip install -r requirements.txt
```

### 🚀 Quick Start

```bash
python vanity_miner_en.py
```

Follow the interactive prompts:
1. Enter target suffix (e.g., `888`, `ABC`)
2. Select generation mode (Mnemonic/Private Key)
3. Select target networks (ETH/TRX/SOL)
4. Select matching mode (Shared/Independent)
5. Wait for generation to complete

### 📖 Usage Guide

#### Generation Modes

| Mode | Description | Speed | Security |
|------|------------|-------|----------|
| Mnemonic | Generate 12-word mnemonic | ⭐⭐ | ⭐⭐⭐⭐⭐ |
| Private Key | Direct random private key | ⭐⭐⭐⭐⭐ | ⭐⭐⭐ |

#### Matching Modes

| Mode | Description | Speed | Result |
|------|------------|-------|--------|
| Shared | One key satisfies all selected networks | Slower | 1 key set |
| Independent | Generate separately for each network | Faster | N key sets |

### 📊 Probability Reference

#### ETH Address (Hexadecimal)

| Suffix Length | Probability | 8-Core Est. Time |
|--------------|------------|------------------|
| 1 digit | 1/16 | <1 second |
| 2 digits | 1/256 | <1 second |
| 3 digits | 1/4,096 | ~5 seconds |
| 4 digits | 1/65,536 | ~1 minute |
| 5 digits | 1/1,048,576 | ~20 minutes |
| 6 digits | 1/16,777,216 | ~5 hours |

#### TRX/SOL Address (Base58)

| Suffix Length | Probability | 8-Core Est. Time |
|--------------|------------|------------------|
| 1 digit | 1/58 | <1 second |
| 2 digits | 1/3,364 | ~5 seconds |
| 3 digits | 1/195,112 | ~4 minutes |
| 4 digits | 1/11,316,496 | ~4 hours |
| 5 digits | 1/656,356,768 | ~10 days |

> ⚠️ In shared mode, probabilities multiply, significantly increasing time

### 💡 Usage Tips

1. **Recommended Suffix Length**
   - ETH: 3-4 digits are practical
   - TRX/SOL: 2-3 digits are practical

2. **Mode Selection**
   - For speed → Private Key + Independent Mode
   - For unified management → Mnemonic + Shared Mode

3. **Security Recommendations**
   - Run in offline/disconnected environment
   - Immediately handwrite and save after generation
   - Never screenshot or transmit over network
